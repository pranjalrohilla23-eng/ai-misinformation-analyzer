# AI Misinformation Analyzer

An AI powered web app that detects fake news, explains why it may be misleading, and helps users verify claims.

## Features
- Fake news detection using NLP models
- Emotional manipulation detection
- Trust score and explanation
- PDF and image analysis (OCR)
- Real time news verification
- Hindi/Hinglish support
- History tracking

## Run locally
pip install -r requirements.txt  
streamlit run app.py
